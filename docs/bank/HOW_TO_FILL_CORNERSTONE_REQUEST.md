# How to fill Cornerstone’s document request

The IJB **Commercial Financial Statement Package** lists:

1. IJB Startup SOI  
2. IJB Startup Balance Sheet  
3. 549 Indian Church SOI  
4. 549 Indian Church Balance Sheet  
5. Durilla 561 Main Pro Forma  
6. **Supporting Documents** ← this was empty; the fill form completes it  

The 549 Balance Sheet also points to: **Bank Statements**, **Personal Financial Statement**, and **Mortgage Statements**.

## Use these files

| Priority | File | What to do |
|----------|------|------------|
| 1 | `Cornerstone_Document_Request_Fill_Form.docx` | Fill every numbered blank (Items 1–22) |
| 2 | `sample_templates/Item_##_*.md` | One sample template per item — copy values in |
| 3 | Attach real PDFs next to each completed item | Bank stmts, PFS, leases, CapEx, etc. |
