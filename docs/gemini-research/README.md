## Working layout assumption (pending measured 1st-floor SF)
**Bank target (Luxe Loft 716):** **9 suites** · **$108,000**/yr suite income.

Until Seneca’s first floor is measured, engineering planning may also use the studied **1,200 sf / 8-suite** prototype:

| Component | Count | SF each | Total SF |
|-----------|-------|---------|----------|
| Micro suites (small) | 5 | 100 | 500 |
| Micro suites (large) | 3 | 150 | 450 |
| Lobby / corridors | — | — | ~250 |
| **Total** | **8 suites** | | **1,200** |

**Scale rule:** suite count ≈ floor SF × (8/1200) after lobby carve-out; or redesign to keep ~20% common + 80% rentable.

## Business model (bank-ready)
1. Acquire 3887 Seneca Street (IJB Enterprises Inc.).
2. Convert first-floor retail to lockable private salon micro-suites for independent beauty operators.
3. Landlord provides walls, power, Wi-Fi, shared lobby/restrooms; wet/vent stubs where required.
4. Multiple suite leases → higher $/SF and lower single-tenant vacancy risk.
5. Two apartments remain leased at **$2,600/mo combined** ($31,200/yr) as baseline income during and after conversion.

## CapEx bands (walls only — from chat; exclude plumbing/HVAC/permits)
| System | Installed $/sf wall face |
|--------|---------------------------|
| Industrial modular | $10–$35 |
| Operable partitions | $15–$110+ |
| Architectural demountable | $45–$110 |
| ALUR-style glass fronts | ~$400/LF entry tier |

Prefer: solid demountable between suites (STC ≥ 40) + glass corridor fronts; sliding doors to save swing clearance in 100 sf rooms.
